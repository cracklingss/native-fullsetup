# React Native Project Creation

## Prerequisites

Before running the project, make sure you have the following installed:

- Node.js
- npm or yarn
- Git
- Expo Go (from PlayStore)

Check your installation:

```bash
node -v
npm -v
git --version
```

## Create a new project

```bash
npx create-expo-app@latest --template default@sdk-56 ./
```
Choose "For learning with Expo Go (SDK 54)"

## Start your Native Project

```bash
npm run start
```
Scan the QR Code from your expo app
